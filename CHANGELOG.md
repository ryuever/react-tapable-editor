# Changelog

## 0.3.0

- Migrated the editor core from Draft.js-era architecture to React 18 + Lexical.
- Added AI-native nodes for context chips, attachment chips, tool chips, artifact blocks, tool-call blocks, and image blocks.
- Added structured prompt payloads with Lexical JSON, portable schema v2, extracted parts, model selection, context, attachments, and tool mode.
- Added tool-call lifecycle fields, action buttons, logs, outputs, run ids, and external invocation ids.
- Added selected block toolbars for delete, duplicate, move, copy JSON, and image editing.
- Added slash menu, context picker, attachment upload adapter, prompt history, and model selector.
- Added Vite library build, declaration output, Playwright e2e tests, and VitePress wiki docs.
